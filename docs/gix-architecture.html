GIX — Complete Technical Architecture

A 15-phase AI pipeline that transforms a Jira ticket into verified, tested React code. 18 specialized agents, 25+ skills, 12 guardrail categories, incremental state management, and multi-layer verification. Accepts Figma, v0.dev prototypes, or screenshots as design input.

The orchestrator (commands/gix.md) is the brain. It never generates code or reads codebases directly. Instead, it spawns specialized agents for each phase, validates their output via gate checks, and manages state for resume safety. Agents read skills for coding patterns and reference data, and communicate through JSON files in the workspace.

Agent Communication Model

Agents are isolated — they don't call each other directly. Communication happens through JSON files in the workspace. The orchestrator reads each agent's output and spawns the next agent with the right context.

sequenceDiagram participant O as Orchestrator (gix.md) participant A1 as Agent N participant FS as Workspace (filesystem) participant SK as Skills (read-only) participant A2 as Agent N+1 O->>O: Check prerequisites (artifact exists?) O->>O: Update pipeline_state.json (in_progress) O->>A1: Spawn with: workspacePath, inputs A1->>SK: Read shared-rules, guardrails, cap-ui-library A1->>FS: Read input artifact (JSON) A1->>A1: Process (generate/analyze/verify) A1->>A1: Run exit checklist A1->>FS: Write output artifact (JSON) A1-->>O: Return summary O->>FS: Read output artifact O->>O: Run gate check (validate output) O->>O: Update state, journal, memory, approach log O->>O: Update live dashboard O->>A2: Spawn next agent Note over O,A2: This cycle repeats for all 15 phases

Circuit breaker: After 3 rework cycles on the same phase pair, the pipeline stops and presents the conflict to the developer with concrete options. It never spins indefinitely.

This reduces human interruptions. Product and codebase questions are resolved by agents before falling back to the user. Only true human-intent questions (preferences, priorities, scope) go to the user directly.

LevelLabelRangeAgent BehaviourEvidence Required
C1Speculative<20%Flag. Don't act alone.Agent reasoning only
C2Plausible20-40%Investigate first.One indirect source
C3Tentative40-60%Act only if reversible.One direct source

Reversibility matrix: Reversible + C4 = act. Irreversible + below C4 = STOP and escalate. This eliminates analysis paralysis while protecting against real damage.
